PART I: BASIC CONCEPTS

Chap1.1: AFFINE SET

Thm1.3

Given $b \in \mathbb R^m$ and $m\times n$ real matrix $B$, the set
$$
M = { x\in\mathbb R^n | Bx=b }
$$
is an affine set in $\mathbb R^n$. Moreover, every affine set may be represented in this way.

Proof

$Bx =b$, $X= {x\in\mathbb R^n | Bx=0}$. We have $M=x+X$.

affine transformation $T((1-\lambda)x+\lambda y) = (1-\lambda)Tx + \lambda Ty$.

Thm1.5

The affine transformations from $\mathbb R^n$ to $\mathbb R^m$ are the mappings $T$ of the form $Tx=Ax+a$, where $A$ is a linear transformation and $a\in\mathbb R^n$.

Corollary 1.6.1

Let $M_1$ and $M_2$ be any two affine sets in $\mathbb R^n$ of the same dimension. Then there exists a one-to-one affine transformation $T$ of $\mathbb R^n$ onto inself such that $TM_1=M_2$.

Chap1.2: CONVEX SETS AND CONES

Thm 2.2

A subset of $\mathbb R^n$ is convex if and only if it contains all the convex combinations of its elements.

Thm 2.7

Let $K$ be a convex cone containing 0. Then there is a smallest subspace containing K, namely
$$
K-K = {x-y|x\in K,y\in K} = \text{aff}~ K
